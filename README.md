## Overview

This mod restores classic zombie apocalypse rules inspired by traditional zombie canon and George A. Romero films.

If the brain is destroyed — the enemy dies.

A single clean headshot will instantly kill:

- Standard Zombies
- Zombie Dogs
- Lickers

Supported weapon types:

- Pistols
- Shotguns
- Submachine Guns
- Magnums
- Minigun

Only true ballistic firearms trigger instant headshot kills.

## Installation

1. Install REFramework:
   https://github.com/praydog/REFramework

2. Download this mod archive.

3. Extract the archive directly into your game directory  
   (the folder where the game .exe file is located).

4. Launch the game.

Make sure REFramework is properly installed before using this mod.

---

## Head Explosion Behavior

Head explosion chance is unchanged.

The probability is identical to vanilla.  
This mod does not alter gore mechanics — only lethality logic.

---

## Exclusions

The following enemies are intentionally excluded:

- Regenerators
- Plant-based enemies (Ivies)
- Special / non-ballistic weapons

Regenerators are excluded to preserve gameplay tension and intended mechanics.

---

## Design Philosophy

This mod focuses on consistency, not difficulty reduction.

Enemy diversity, boss fights, and overall pacing remain intact.

The goal is to reinforce internal outbreak logic:

If the brain is destroyed — the threat is neutralized.

Hardcore difficulty is recommended for the best gameplay balance.
